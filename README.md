# web2-template

To start the project

1. copy `backend/.env.temp` to `backend/.env` and fill in the data in `backend/.env`

```bash
cp backend/.env.temp backend/.env
```

2. And run

```bash
npm install
npm start
```

> Dev mode:
>
> ```bash
> npm run dev
> ```
